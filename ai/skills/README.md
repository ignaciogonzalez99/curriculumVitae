# AI Skills

Esta carpeta contiene guias reutilizables para que Codex pueda actuar con distintos perfiles de apoyo segun la tarea. Cada skill define proposito, situaciones de uso, comportamiento esperado, workflow, formato de salida, checklist y guardrails.

## Skills creadas

- `$frontend-dev`: usar para tareas de frontend senior con React, TypeScript, Next.js, Tailwind, CSS responsive, componentes, accesibilidad basica, UX/UI, refactors, debugging visual e integracion con APIs.
- `$cv-analysis`: usar para analizar, mejorar y adaptar CVs, detectar puntos debiles, mejorar logros, ordenar experiencia e identificar keywords ATS.
- `$writing`: usar para mejorar redaccion, emails, documentos profesionales, prompts, documentacion tecnica, tono, gramatica y traducciones cuando se pidan.
- `$psychoanalytic-reflection`: usar para reflexion personal inspirada en psicoanalisis, exploracion emocional, patrones de pensamiento, vinculos y escritura introspectiva. No reemplaza terapia ni diagnostica.
- `$hr-career-strategy`: usar para estrategia profesional, empleabilidad, LinkedIn, entrevistas, analisis de ofertas, brechas de skills, aprendizaje y negociacion salarial general.

## Ejemplos de invocacion

- "Usa $frontend-dev para refactorizar este componente y mejorar su responsive."
- "Usa $cv-analysis para revisar mi CV y adaptarlo a esta oferta laboral."
- "Usa $writing para mejorar este email y hacerlo profesional pero cercano."
- "Usa $psychoanalytic-reflection para ayudarme a ordenar lo que siento sobre esta situacion."
- "Usa $hr-career-strategy para crear un plan de 4 semanas para conseguir mejores oportunidades."

## Mantenimiento

Mantener estas skills actualizadas segun evolucione el perfil profesional, el stack tecnologico, los objetivos laborales y las necesidades de escritura o reflexion. Si una skill empieza a repetirse con instrucciones nuevas, conviene incorporarlas a su `SKILL.md` para que Codex las pueda reutilizar.
